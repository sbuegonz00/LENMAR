WORLD WIDE WEB


En informática, la World Wide Web (WWW) o Red informática mundial comunmente conocida como la web, es un sistema de distribucion de documentos de hipertexto o hipermedios interconectados y accesibles vía Internet. Con un Navegador web, un usuario visualiza sitios web compuestos de páginas web que pueden contener, texto, imagenes, vídeos u otros contenidos multimedia, y navegar a través de hypervinculos

Historia

La web se desarrollo entre marzo de 1989 y diciembre de 1990 por el inglés Tim Berners-Lee con la ayuda del belga Robert cailliau mientras trabajaban en el CERN en Ginebra, Suiza, y publicado en 1992.

Desde entonces, Berners-Lee ha jugado un papel activo guiando el desarrollo de estándares web (como los lenguajes de marcado con los que se crearan las páginas web), y en los últimos años ha abogado por su vision de una web semántica.
